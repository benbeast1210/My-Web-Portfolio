Decoder Ring

This project is an exercise in classical coding challenges. The Caesar Cipher, the Polybius Square, and the Substitution Cipher. I used many higher-order functions including .reduce(). I also built this project with Test-Driven Development in mind. My goals were to first pass tests, then refactor as necessary.
